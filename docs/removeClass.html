    removeClass

    removeClass should take collections and remove the passed class name on any node that has it

    • memberof class api
    • arg string :: class name to be removed across all nodes in collection.
    • returns modified collection

    example

    <div class="example test">Hello World</div>

    $('.example').removeClass('test')

    <div class="example">Hello World</div>

    (function($) {
  $.plugin('removeClass',function(klass) {
    this.each((node) => {
      node.className = node.className.replace(klass, '');
    })

    return this;
  });
})(collector)
